# This is my website repo.

1. Clone the repo using:
    ```bash
    git clone https://github.com/Tiesco789/tiesco-site.git
    ```
2. Go to the folder and run the following command to install dependencies:
    ```bash
    npm install
    ```
3. Run the projecto locally:
    ```bash
    npm run start
    ```
4. THATS ALL FOLKS. HOPE YOU LIKE IT
